Restar dos horas

Titulo:Restar dos horas
Lenguaje:PHP
Este código es una función que resta dos horas
Votar [1.00 / 1 votos] - Visitas 23.183
Comentarios: 0
NOTA: Tienes que enviar a la función la hora inicio y final.
Ejemplo:
$horaini="10:05:20";
$horafin="14:05:20";
echo RestarHoras($horaini,$horafin);

<? # http://www.lawebdelprogramador.com # tiene que recibir la hora inicial y la hora final function RestarHoras($horaini,$horafin) { $horai=substr($horaini,0,2); $mini=substr($horaini,3,2); $segi=substr($horaini,6,2); $horaf=substr($horafin,0,2); $minf=substr($horafin,3,2); $segf=substr($horafin,6,2); $ini=((($horai*60)*60)+($mini*60)+$segi); $fin=((($horaf*60)*60)+($minf*60)+$segf); $dif=$fin-$ini; $difh=floor($dif/3600); $difm=floor(($dif-($difh*3600))/60); $difs=$dif-($difm*60)-($difh*3600); return date("H-i-s",mktime($difh,$difm,$difs)); } ?>
